Banner for specific domains
======
Specify domains (e.g: `*.dev|*.local`) for the banner to appear:

![banner](banner-screenshot.png)
![settings](settings-screenshot.png)

Usage:
------
Drop `banner.crx` to your Chrome extensions page.
